Simulation on Law of Large Numbers

In the probability theory the Law of the Large Numbers states that when do you repeat an experiment a large number of times, the average of the results of experiment should be close to the expected value. The sample average will become closer as more trials are performed.

To illustrate the intuition behind this law we will consider our experiment to be a run of rolls of a dice. A dice has 6 possible outcomes, the integer numbers from 1 to 6, with each output having equal probability. Therefore teh expected value of a single die roll is \( (1+2+3+4+5+6)/6=3.5 \)

We simulate the event of a single die roll to be a draw of a number from the discrete uniform distribution with minimum value equals to 1 and maximum value equals to 6. To simulate a large number of independent events we simply draw a large sample of generated random numbers from the same distribution.

Rapaio makes this possible by using the following code:

        final int N = 1_000;
        Vector dice = new DUniform(1, 6).sample(N);

Thus we have stored in a vector N (1000) outputs of those events. We compute the running mean using OnlineCoreStat:

        OnlineCoreStat ocs = new OnlineCoreStat();
        Vector mean = new NumericVector("mean", N);
        for (int i = 0; i < events.getRowCount(); i++) {
            ocs.update(events.getValue(i), 1);
            mean.setValue(i, ocs.getMean());
        }

Now we have the running mean stored in the vector mean and we can plot how that running mean evolves as the size of the sample grows.

Thus we can clearly notice two fact from the plot above. First fact is that the running average gets closer to the expected value, as sample size grows. Second fact is that deviation from expected value is smaller as the sample size grows aka. smaller variation.

The code for drawing the plot follows:

        Plot plot = new Plot();

        ABLine abline = new ABLine(plot, 0, 3.5);
        abline.opt().setLwd(2);
        abline.opt().setColorIndex(new OneIndexVector(1));
        plot.add(abline);

        Lines lines = new Lines(plot, new IndexVector("x", 1, N, 1), mean);
        lines.opt().setLwd(2);
        lines.opt().setColorIndex(new OneIndexVector(2));
        plot.add(lines);

        plot.getOp().setYRange(2.5, 4.5);
        draw(plot, 800, 300);
